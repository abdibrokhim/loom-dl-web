# Contributing to Loom Video Downloader

Thanks for your interest in contributing to Loom Video Downloader! This document provides guidelines and instructions for contributing.

## Getting Started

### Prerequisites
- Node.js >= 18
- npm >= 10

### Setup

1. [Fork](https://github.com/abdibrokhim/loom-dl-web) and clone the repository:

```shell
git clone https://github.com/YOUR_USERNAME/loom-dl-web.git
cd loom-dl-web
```

2. Install dependencies using npm:

```shell
npm install
```

3. Start the development server:

```shell
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
